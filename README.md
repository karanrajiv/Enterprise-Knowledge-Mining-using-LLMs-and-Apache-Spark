# Enterprise Knowledge Mining using LLMs and Apache Spark

This project demonstrates how to use Apache Spark for large-scale data preprocessing and integrate it with a Large Language Model (LLM) to mine insights from enterprise documents.

## Features
- Batch processing of documents using Apache Spark
- Named Entity Recognition and text classification using Hugging Face Transformers
- Searchable document embeddings using FAISS

## Tech Stack
- Python
- Apache Spark
- Hugging Face Transformers
- FAISS
- Streamlit (optional for UI)

## Usage
1. Preprocess documents with Spark
2. Generate embeddings with LLM
3. Store and index embeddings in FAISS
4. Run inference queries to retrieve insights
